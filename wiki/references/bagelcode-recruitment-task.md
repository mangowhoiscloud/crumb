## 1. 과제 목표 (verbatim)

> AI 코딩 에이전트(Claude Code, Codex, Gemini CLI 등)를 사용하여, 여러 AI 에이전트가 서로 통신하며 사용자와 협업할 수 있는 도구를 만드세요.

핵심 동사 3개:
1. **여러 AI 에이전트가 서로 통신** — 에이전트 ↔ 에이전트
2. **사용자와 협업** — 에이전트 ↔ 사람
3. **AI 코딩 에이전트를 사용하여 (만든다)** — 메타: 도구 자체를 에이전트로 만든다

## 2. 과제 조건 (verbatim)

- 두 개 이상의 AI 에이전트가 메시지를 주고받을 수 있어야 합니다
- 사용자가 이 협업 과정에 개입하거나 관찰할 수 있어야 합니다
- 통신 방식, 프로토콜, UI, 언어·프레임워크는 자유입니다
- AI 코딩 에이전트를 사용하여 개발해야 합니다
- README대로 실행했을 때 동작해야 합니다

## 3. 제출 안내 (verbatim)

- **제출 형식**:
  - 작동하는 코드 (GitHub repo 또는 .zip)
  - 개발 과정에서 에이전트에 활용한 `.md` 파일 포함 ← **CLAUDE.md / AGENTS.md / SKILL.md 등 메타 인스트럭션 제출 명시**
  - 에이전트 세션 로그 (JSONL) **또는** 화면 녹화

## 4. 유의사항 (verbatim)

- 사용 도구 제한 없음 (Claude Code, Cursor, Codex 등 자유)
- 기한 내 1회만 응시 / 제출 후 수정 어려움
- 일정 조정 필요 시 채용팀 문의
